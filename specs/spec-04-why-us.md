# Spec 04: Why Choose Us Section (Dlaczego My)

## Target File
`src/sections/WhyChooseUs.tsx`

## Impeccable Skills
- `/impeccable:clarify` - Card headlines and descriptions
- `/impeccable:layout` - Card arrangement

## Design Context
- Trust at first glance
- Warmth without softness
- No side-stripe borders, no nested cards

---

## Changes Required

### Section Header
**Current**: "Co nas wyróżnia"
**Keep as is** (or minor adjustment if needed)

### Cards (4 total)

#### A. Gwarancja rzetelności
**Title**: Gwarancja rzetelności

**Description**:
```
Twoje bezpieczeństwo finansowe jest naszym priorytetem. Dbamy o najwyższe standardy pracy i bierzemy pełną odpowiedzialność za poprawność Twoich rozliczeń. Dzięki naszej skrupulatności zyskujesz solidny fundament i absolutny spokój o przyszłość swojej firmy.
```

#### B. Bezpośrednia współpraca
**Title**: Bezpośrednia współpraca

**Description**:
```
Stawiamy na relację, a nie tylko na liczby. Pracujesz z ludźmi, którzy naprawdę znają Twoją branżę i są dostępni zawsze wtedy, gdy potrzebujesz merytorycznego wsparcia. Rozumiemy Twoje potrzeby i mówimy Twoim językiem, budując partnerstwo oparte na zaufaniu.
```

#### C. Szybkość i precyzja działania
**Title**: Szybkość i precyzja działania

Szanujemy Twój czas, dlatego gwarantujemy szybkie rozliczenie dokumentów. Dzięki sprawnej obsłudze zawsze masz dostęp do aktualnych danych finansowych, co pozwala Ci trzymać rękę na pulsie i nigdy nie przegapić ważnego terminu.
```

```

## Visual Notes
- Icons should be simple, purposeful (not decorative)
- Cards should have equal visual weight
- No "Gwarancja jakości" with shield icon clichés
- Consider removing icons entirely if they don't add meaning
