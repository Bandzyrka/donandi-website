- "misję społeczną" visually emphasized (brand color)

### B. Description Text
**New**:
```
Nasza historia z księgowością zaczęła się w 2006 roku. Przez blisko dwie dekady zbieraliśmy doświadczenie, które dziś pozwala nam oferować wsparcie na najwyższym poziomie. Specjalizujemy się nie tylko w obsłudze przedsiębiorstw, ale jesteśmy również ekspertami w unikalnym obszarze księgowości NGO oraz Przedsiębiorstw Ekonomii Społecznej (PES).

Naszą misją jest zdejmowanie ciężaru biurokracji z barków tych, którzy zmieniają świat – zarówno biznesowy, jak i społeczny. Jako zespół certyfikowanych księgowych działamy pod patronatem Fundacji Donandi, co jest dla Ciebie gwarancją etyki, transparentności i merytorycznej niezawodności.
```

**Links to emphasize**:
- "księgowości NGO" - link to NGO service card
- "Przedsiębiorstw Ekonomii Społecznej (PES)" - link to PES service card
- "Fundacji Donandi" - external link to foundation website
